# LocateMe
A tool to geolocate a computer by IP address.

## Requirements
* Install [Node.js](https://nodejs.org/en/download/)

* npm install request

* npm install web-terminal

## To run
* In terminal: node http_srv.js

* In browser go to site: [127.0.0.1:6067](http://127.0.0.1:6067)
